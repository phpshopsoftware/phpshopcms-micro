Добро пожаловать

PHPShop.CMS – уникальный проект Рунета, позволяющий любому пользователю Интернет самостоятельно и без вложений создать и поддерживать сайт любой сложности. В Вашем распоряжении находится готовая платформа для создания сайта и набор модулей, с помощью которых можно настроить функционал сайта под свои нужды.

Система управления сайтом PHPShop CMS и все сопутствующие программы распространяются бесплатно, никаких скрытых платежей не предусмотрено.

Использование

PHPShop.CMS Micro - это специальная облегченная версия PHPShop.CMS, для работы которой, не требуется база данных MySQL. Вся информация хранится в статичных HTML файлах на сервере. Хорошо подходит для создания многостраничного сайта или "Landing page" с блочной структурой на бесплатных хостингах. Имеет большое количество бесплатных шаблонов дизайна на любой вкус и цвет.

Системные требования

PHPShop.CMS Micro работает на любом хостинге с установленными Apache или Nginx, PHP 5.6+.
